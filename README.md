# call2word

Расшифровка видеозвонков в русский текст + готовый промт для LLM + отчёт в Word.

Оптимизирован под **Яндекс Телемост** (автоопределение папки, таблица с датой/временем/длительностью), но работает с любыми созвонами и видеофайлами — Zoom, Teams, Google Meet, локальные записи.

Работает полностью офлайн на CPU. Не отправляет данные никуда.

## Что делает

- Автоматически находит папку `Documents\Телемост` при запуске
- Показывает встречи таблицей: название, дата, время, длительность
- Деноизинг аудио перед транскрипцией (highpass + noisereduce) — улучшает качество распознавания
- Транскрипция через `faster-whisper` (модель `large-v3-turbo`, CPU/int8)
- Собирает готовый промт по шаблону с `{text}` и отправляет в LLM
- Принимает JSON-ответ от LLM и сохраняет структурированный Word-документ
- Все файлы сохраняются в подпапку `Отчёты/<имя записи>/` рядом с видео

## Запуск на Windows

```
setup_portable.bat   ← один раз, скачивает Python и зависимости
run.bat              ← запуск приложения
```

## Запуск на macOS / Linux

```bash
pip install PySide6 faster-whisper imageio-ffmpeg noisereduce soundfile scipy
python main.py
```

## Использование

1. Приложение автоматически открывает папку Телемоста — или нажмите **Открыть папку**
2. Отметьте галочкой нужную запись (по умолчанию — последняя)
3. Нажмите **Запустить** — модель скачается при первом запуске (~1.6 ГБ)
4. После расшифровки скопируйте промт и отправьте в любую LLM
5. Вставьте ответ LLM во вкладку **Ответ LLM → Word** и нажмите **Сохранить Word**
6. Папка с файлами откроется автоматически

## Модели

По умолчанию `large-v3-turbo` — оптимальный баланс для русского языка на CPU.

| Модель | RAM | Диск | Качество |
|---|---|---|---|
| `large-v3-turbo` | ~2-3 ГБ | ~1.6 ГБ | отличное, быстрее large |
| `large-v3` | ~4-5 ГБ | ~3 ГБ | максимальное |

Модели скачиваются из HuggingFace при первом запуске и кэшируются в `~/.cache/huggingface/`.

## Формат ответа LLM

Шаблон просит модель вернуть JSON:

```json
{
  "title": "название созвона",
  "short_summary": "резюме в 3-5 предложениях",
  "blocks": [
    {
      "number": 1,
      "title": "название блока",
      "text": "что обсуждали",
      "decisions": [],
      "tasks": [{"who": null, "what": "задача", "deadline": null}],
      "open_questions": [],
      "risks": []
    }
  ],
  "global_tasks": [],
  "unclear_or_risky": []
}
```

Если LLM вернёт обычный текст — он тоже сохранится в Word без разбора.

## Частые проблемы

- **Модель не скачивается** — нужен интернет при первом запуске, доступ к huggingface.co
- **Нехватка RAM** — переключитесь на другую модель или закройте другие программы
- **Долго обрабатывает** — норма для CPU; двухчасовой созвон на `large-v3-turbo` ~30-60 мин
- **Не открывается папка Телемоста** — нажмите **Открыть папку** вручную
